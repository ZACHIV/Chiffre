# Chiffre 产品战略分析

## 第一性原理：这款应用解决了什么根本问题？

数字听力难，不是因为"没练习过"，而是因为**L1 干扰**（母语数字已深度固化，外语数字无法绕过大脑去自动处理）加上**认知负荷**（法语四舍五入进位系统要求实时心算）。

这是一个**有明确认知科学依据的特殊瓶颈**，与"听力理解能力弱"是两件不同的事。

---

## 市场定位选择

### 选项 A：全面听力练习器 ❌

竞品是 Duolingo、Pimsleur、Babbel。它们拥有：
- 数亿美元投入的内容库
- 专业录音演员和语言学顾问
- 完善的 SRS 算法

**Chiffre 在这条赛道上没有结构性优势，也无法赢。**

### 选项 B：守住数字，但做深做精 ⚠️

调研发现市场上的数字类应用（Numfred、ReflexNum）都相当初级，功能单一。**这个细分市场有真实需求但缺少高质量产品。**

Chiffre 已有的优势：双语、场景化（价格/电话/火车/航班）、自适应语速、句子语境。

**但"数字"这个领域本身的天花板有限——用户练完了怎么办？**

### 选项 C（推荐）：扩展到「结构化实用听力」这个细分品类 ✅

数字之所以是瓶颈，是因为它是**高频出现、格式可预测、但认知处理难以自动化**的内容。这个特征不只属于数字：

| 内容 | 为何是瓶颈 | 实用场景 |
|------|-----------|---------|
| 数字（已有） | L1干扰 + 心算负荷 | 价格、时间、电话、交通 |
| **字母拼读** | 法语字母发音非直觉（H comme Henri）| 拼名字、报邮箱、导航菜单 |
| **地址门牌** | 数字+街道名混合，语速快 | 打车、酒店、快递 |
| **百分比/统计** | 数字嵌入新闻句子 | 看新闻、听广播 |
| **体育比分** | 快速报分，含队名 | 球赛转播 |
| **火车/航班广播** | 站台噪音、快速播报 | 出行（已有雏形）|

这些内容的共同点：**不需要理解复杂语法，只需要自动化处理特定语言块。** 这正是 Chiffre 的核心能力。

---

## 调研数据支持

### 市场认证

- **Numfred**、**ReflexNum** 等专业数字应用的存在证明了市场需求
- 调查显示学习者在流畅对话中遇到数字时"完全断路"——说明这是独立的学习瓶颈
- 法语数字系统的特殊性（vigesimal）使其成为通用应用无法深度解决的问题

### 教学科学支持

研究表明最有效的听力学习方式：
- **结构化练习**（预听→首听→验证）—— Chiffre 的核心流程
- **格式可预测的内容**进行集中深度练习 —— 数字、字母拼读
- **主动处理策略**（自我监控、输入、反馈）—— 当前已实现
- **快速处理能力**通过重复建立自动化 —— 自适应速度机制

## 具体可改进方向（按优先级）

### P0: 真实音频替代 TTS 🎯

**当前最大的迁移障碍。** TTS 与真实场景差异太大。

改进方式：录制一套多口音、多语速的真实音频包，覆盖所有现有模式。即使只有 200-300 条，体验质变。

真实场景示例：
- SNCF 车站广播（法语）
- 法国广播电台天气预报中的温度/日期
- 电话接线员报号码

### P1: 字母拼写模式（Épellation）🆕

法语电话客服、机场登机口、酒店 check-in 都大量使用 NATO 式字母拼读：
```
"Votre réservation, c'est A comme Anatole, B comme Bernard..."
```

**市场机会**：对中国学习者极难，且在市面上**完全没有专门练习工具**。

实现方式：
- 8 种法语字母表示法（Anatole, Berthe, Céline...）
- 发音拼读 → 用户写出字母 → 验证
- 进阶：快速拼读组合（"SNCF"、"RATP"等常见缩写）

### P2: 错题本 + 间隔重复

把答错的内容放入复习队列，以 Leitner 箱子逻辑安排复习时机。

**当前最大的算法空白**——随机出题的效率远低于基于遗忘曲线的出题。

实现方式：
- 记录每题答题历史（首次、错误次数、最后正确时间）
- 基于遗忘曲线计算下次复习时机
- 优先推送"学过但容易忘"的内容

### P3: 速度解锁可视化

当前连对提速是隐形的。改为可见的「解锁」体验：

```
第一关：0.42 慢速，仅基础数字
第二关：0.47，解锁价格/时间
第三关：0.52，解锁复杂格式（火车、航班）
第四关：0.57，真实语速全模式
```

**产品叙事升级**：Chiffre = 从听不懂到能跟上真实法语的训练器。

### P4: 听写模式（Dictée）

不只识别数字，而是听写整句：

```
播放：         "Le prochain train pour Lyon part à quatorze heures
              trente, quai numéro six."
用户输入示例：  14h30, quai 6
            或 lyon 14:30 quai 6
```

这把碎片化练习升级为接近真实听力任务的综合练习。

### P5: 口音包

相同内容，切换不同口音：
- 法语：巴黎 / 魁北克 / 南法
- 西班牙语：西班牙 / 拉丁美洲

**目标用户**：中高级学习者，对标北美和欧洲大陆的不同口音。

## 推荐产品路线图

### Phase 1（当前 + 1-2 个月）
- ✅ 完成数字模式优化
- 新增：字母拼读基础模式（拉丁字母表）
- 新增：错题间隔重复算法

### Phase 2（2-3 个月）
- 速度解锁系统（可视化进度）
- 真实音频包 v1（数字 + 常见场景）
- 听写模式（易模式：多选 → 难模式：自由输入）

### Phase 3（3-6 个月）
- 完整字母拼读模块（法语 NATO 表 + 常见缩写）
- 口音包 v1（巴黎 vs 蒙特利尔）
- Web 版本增强（离线模式）

## 最终定位语句

**Chiffre 是为法语/西班牙语学习者设计的结构化听力专项工具。**

目标：帮助能进行日常对话的学习者，突破"听到数字和结构化信息就断路"这一具体障碍。

特点：
- 💪 垂直深度：不做全科，专注数字 + 结构化实用信息
- 🎯 痛点精准：解决认知科学级别的学习瓶颈
- 🚀 可扩展：从数字→字母→地址→广播，形成生态
- 🌍 多语言：法语（含魁北克）+ 西班牙语（含拉美）
- ⚡ 快速反馈：适合碎片时间练习，自适应难度
