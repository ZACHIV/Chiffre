# 语音选项简化更新

## 更新日期
2026-01-06

## 🎙️ 变更说明

### 移除的语音
- ❌ Daniel（男声·备选）- 非法语口音
- ❌ Marie（女声·备选）- 非法语口音

### 保留的语音
- ✅ **Thomas（男声）** - 纯正法语口音
- ✅ **Amélie（女声·推荐）** - 纯正法语口音，苹果翻译 App 使用

## 📊 最终语音列表

| 语音 | 性别 | 口音 | 推荐度 | 说明 |
|------|------|------|--------|------|
| **Thomas** | 男 | 🇫🇷 法语 | ⭐⭐⭐⭐ | 标准法语男声 |
| **Amélie** | 女 | 🇫🇷 法语 | ⭐⭐⭐⭐⭐ | 高质量法语女声（默认） |

## 🎯 为什么只保留这两个？

1. **纯正法语口音**
   - Thomas 和 Amélie 都是标准的法国法语发音
   - 适合法语学习者练习听力

2. **性别选择**
   - 保留一男一女，满足不同偏好
   - 可以适应不同发音风格

3. **质量保证**
   - Amélie 是 Enhanced 质量，发音自然清晰
   - Thomas 是 Compact 质量，系统自带

## 🔧 技术变更

### 修改的文件
- `SpeechManager.swift`
  - 从 `FrenchVoice` 枚举中移除 `daniel` 和 `marie`
  - 简化了 `displayName` 和 `icon` 的 switch 语句

### 代码对比

#### 之前（4个语音）
```swift
enum FrenchVoice: String, CaseIterable, Identifiable {
    case thomas = "Thomas"
    case amelie = "Amélie"
    case daniel = "Daniel"    // 已移除
    case marie = "Marie"      // 已移除
}
```

#### 现在（2个语音）
```swift
enum FrenchVoice: String, CaseIterable, Identifiable {
    case thomas = "Thomas"
    case amelie = "Amélie"
}
```

## 📱 用户界面变化

### 设置界面
现在语音选择区域只显示两个选项：

```
Voix (语音)                                          [🔊 试听]
┌──────────────────┬──────────────────┐
│ Thomas (男声)     │ Amélie (女声·推荐) │
└──────────────────┴──────────────────┘
```

更加简洁明了！

## ✅ 优点

1. **界面更简洁** - 只有两个选项，不会让用户困惑
2. **选择更明确** - 男声/女声各一个，清晰直观
3. **质量有保证** - 都是纯正法语口音
4. **减少维护** - 代码更简单，更容易维护

## 🎓 推荐使用

- **默认推荐**: Amélie（女声）
  - 发音清晰自然
  - 苹果翻译 App 同款
  - 适合大多数学习者

- **备选选择**: Thomas（男声）
  - 标准法语男声
  - 系统自带，无需下载
  - 适合偏好男声的学习者

## 📝 注意事项

- Amélie 可能需要在系统设置中下载
- 路径：设置 → 辅助功能 → 朗读内容 → 语音 → 法语
- 如果未下载，系统会自动使用默认法语语音
